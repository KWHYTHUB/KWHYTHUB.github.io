How to Use & Where to Find Mods

After installing Sapphire, you should see a new button for it in the bottom bar of the main menu. This button leads to a menu where you can configure all your mods & install new ones.

The default tab here is the list of installed mods. Press View to see more info about each mod.

You can download new mods from the Download page. Press View to see info about the mod and begin installation.

Need help?

If you need help using Sapphire, you can open an issue on Github or report to us on Discord. Please note that issues using mods should be directed to the mods' developers instead!
